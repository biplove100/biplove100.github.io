# E-commerce Product Display Project

## Overview
This project is an e-commerce product display website that fetches product data from a Shopify API and dynamically displays products based on user-selected categories. It allows users to choose between Men, Women, and Kids categories to view corresponding products.

## Features
- Fetches product data from the Shopify API.
- Displays products dynamically based on category selection.
- Supports responsive design for various screen sizes.
- Provides product details such as image, title, vendor, price, and discount information.
- Allows users to add products to their cart.

## Technologies Used
- HTML: Markup language used for structuring the website's content.
- CSS: Stylesheet language used for styling the website's appearance.
- JavaScript: Programming language used for dynamic behavior and interactivity.
- Fetch API: Used for making asynchronous HTTP requests to fetch product data from the Shopify API.

## Setup
1. Clone the repository to your local machine:
    git clone https://github.com/biplove100/biplove100.github.io.git

2. Open the `index.html` file in your web browser to view the website.

## How to Use
1. Upon opening the website, users are presented with a selection of categories: Men, Women, and Kids.
2. Click on a category button to view products corresponding to that category.
3. The products will be displayed below the category selection in a card layout, showing product images, titles, vendors, prices, and discount information.
4. Users can click the "Add to Cart" button to add a product to their cart.

## Folder Structure
- `index.html`: HTML file containing the structure of the website.
- `style.css`: CSS file containing styles for the website's appearance.
- `script.js`: JavaScript file containing the logic for fetching product data and rendering products on the website.
- `README.md`: Markdown file containing information about the project, setup instructions, and usage guidelines.
